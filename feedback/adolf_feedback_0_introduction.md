## Назначение

Модуль «Правки» — внутренняя система обратной связи для сбора замечаний, пожеланий и багрепортов от сотрудников по всем модулям системы ADOLF.

Сотрудники создают запросы, привязанные к конкретному модулю (Контент-Фабрика, Репутация, Watcher и др.), а разработчики и администраторы отвечают в комментариях и управляют статусами.

## Ключевые возможности

| Возможность | Описание |
|-------------|----------|
| Разделы по модулям | Администратор создаёт разделы из пресетных модулей или кастомные с иконкой |
| Запросы | Заголовок + описание + привязка к разделу + статус |
| Комментарии | Цепочка комментариев под каждым запросом с аватарками |
| Архив | Перемещение старых запросов в архив с подтверждением |
| Роли | Настройка доступа через админ-панель |
| Иконки | Пресетные модули с SVG-иконками, кастомные с lucide icon picker |
