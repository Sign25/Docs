**Модуль:** Правки (Feedback)

## Структура страницы

| Элемент | Описание |
|---------|----------|
| Хедер | Иконка + «Правки» + BETA бейдж + шестерёнка (админ) + «Новая правка» |
| Табы разделов | Кнопки модулей с иконками + «Архив» (только админ) |
| Фильтры | Статус (все/новые/в работе/готово/отклонённые) + счётчик |
| Таблица | Автор (аватар+имя), Запрос (заголовок+текст+последний комментарий), Статус, Дата, Действия (админ) |
| Раскрытие | Клик на строку → комментарии + поле ввода |

## Табы разделов

- Каждый раздел — кнопка с иконкой модуля
- Активный раздел выделен (variant=default)
- «Архив» — крайний правый, только для админов, иконка Archive
- При переключении на архив — сбрасывается фильтр раздела, появляется фильтр «Все разделы»

## Управление разделами (админ)

Шестерёнка в хедере → панель:
- Список существующих разделов с иконками, кнопками редактирования и удаления
- Создание: два режима
  - **«Из модулей»** — выпадающий список пресетных модулей (уже добавленные скрыты), иконка подтягивается автоматически
  - **«Свой раздел»** — кнопка icon picker (сетка lucide SVG 8×N) + текстовое поле названия

## Icon Picker

- Кнопка `+` с выбранной иконкой
- По клику — выпадающая сетка 8 колонок с lucide SVG иконками
- 20+ иконок: message-square, bug, sparkles, settings, zap, star, bell, shield, globe, code, file-text, package, users, flag, tag, rocket, trending-up, layers, lock, eye, search, heart, calendar
- Клик на иконку — выбор, закрытие picker

## Таблица запросов

| Колонка | Описание |
|---------|----------|
| Автор | Аватар (size-7, rounded-full) + имя |
| Запрос | Заголовок (font-medium) + превью текста (line-clamp-1) + последний комментарий (аватар+имя+текст) + счётчик комментариев |
| Статус | Badge с цветом: new=синий, in_progress=жёлтый, done=зелёный, rejected=красный |
| Дата | dayjs fromNow (относительная) |
| Действия (админ) | FilterSelect статуса + кнопка архива + кнопка удаления |

## Комментарии

- Раскрываются при клике на строку таблицы
- Несколько запросов можно открыть одновременно
- Каждый комментарий: аватар + имя + бейдж «Админ» (если role=admin) + дата + текст
- Поле ввода + кнопка отправки (Enter для отправки)
- Комментарии подгружаются при загрузке страницы (для счётчиков)

## Подтверждения

- **Архивация** — ConfirmDialog «Переместить в архив?»
- **Восстановление** — ConfirmDialog «Восстановить из архива?»
- **Удаление** — ConfirmDialog «Удалить правку? Безвозвратно.»
