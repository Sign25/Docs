## Назначение

Модуль обеспечивает внутреннюю коммуникацию между сотрудниками:
- **Коллеги** — личные сообщения (DM) между двумя пользователями
- **Каналы** — групповые чаты с произвольным количеством участников

Построен на инфраструктуре каналов Open WebUI с доработками для ADOLF.

## Ключевые возможности

| Возможность | Описание |
|-------------|----------|
| Личные сообщения | DM между двумя пользователями |
| Групповые каналы | Чаты с несколькими участниками |
| Real-time | WebSocket доставка сообщений |
| Онлайн-статус | Зелёный индикатор с пульсацией |
| Непрочитанные | Счётчик непрочитанных сообщений |
| Реакции | Реакции на сообщения |
| Ответы/треды | Reply to + треды обсуждений |
| Аватары | Фото профиля в списке и чате |
| Скрытие DM | С подтверждением, можно начать заново |

## Расположение в UI

- Сайдбар → секция «Коллеги» (DM, появляется при наличии диалогов)
- Сайдбар → секция «Каналы» (групповые, всегда видна)
- Админка → Пользователи → кнопка «Написать» (MessageSquare)
