## Welcome to ADOLF Documentation

**ADOLF** (Automated Digital Operations and Logistics Framework) is an AI-driven e-commerce automation platform for marketplace operations.

### Modules

- **Core** — User management, Open WebUI, infrastructure
- **Knowledge** — Corporate knowledge base with RAG
- **Reputation** — AI-generated review responses
- **Watcher** — Competitive price monitoring
- **Content Factory** — SEO content generation
- **Marketing** — Automated advertising campaigns
- **Scout** — Predictive analytics for niche analysis
- **CFO** — Financial reporting and analytics
- **Lex** — Legal monitoring
- **Office** — AI agent dashboard
- **Shop** — WooCommerce integration
- **Logistic** — Warehouse optimization
