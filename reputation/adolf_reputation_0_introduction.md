## Что такое ADOLF REPUTATION

ADOLF REPUTATION — функциональный модуль системы ADOLF, обеспечивающий автоматизированное управление репутацией бренда на маркетплейсах.

### Поддерживаемые платформы

| Платформа | Отзывы | Вопросы | Публикация ответов |
|-----------|:------:|:-------:|:------------------:|
| Wildberries | ✅ | ✅ | ✅ |
| Ozon | ✅ | ✅ | ✅ |
| Яндекс.Маркет | ✅ | ✅ | ✅ |

### Основные возможности v2.1

| Функция | Описание |
|---------|----------|
| Автоматический сбор | Polling отзывов и вопросов каждые 5 минут |
| AI-анализ | Классификация тональности и тем |
| RAG-генерация | Персонализированные ответы с данными о товаре |
| Workflow | Утверждение, редактирование, перегенерация |
| Публикация | Автоматическая отправка ответов на маркетплейсы |
| Аналитика | Статистика и отчёты |

## Быстрый старт

### Для менеджера

1. Откройте Open WebUI
2. Выберите агента `@Adolf_Reputation`
3. Напишите: "Покажи новые отзывы"
4. Просмотрите черновики ответов
5. Утвердите или отредактируйте

### Для администратора

1. Убедитесь, что environment variables настроены
2. Проверьте работу Celery Beat: `celery -A app beat`
3. Проверьте работу Workers: `celery -A app worker`
4. Проверьте логи polling
