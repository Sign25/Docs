# ЧЕРНОВИК: Аналитика — Телодвижения vs Охана маркет
*Дата: 31.03.2026*

## 1. ФИНАНСОВЫЕ ПОКАЗАТЕЛИ

### ООО СКВТ (Телодвижения)
- Выручка 2022: 356,6 млн руб. (+251% г/г)
- Прибыль 2022: 43,9 млн руб. (+751%)
- ПРИБЫЛЬ 2024: **630,5 млн руб.** → выручка ~3-4 млрд руб. (оценка)
- Место WB 2023: ТОП-22 из 1,1 млн продавцов

### ООО Охана маркет
- Выручка 2024: **2,612 млрд руб.**
- Прибыль 2024: 222,9 млн руб. (маржа 8,5%)
- WB данные (90 дн.): 77 280 продаж, 511 SKU, ср.чек 732 руб.
- % отмен заказов WB: 27,1%
- Комиссия WB (Брюки/Пижамы): 34-38%

## 2. МОДЕЛЬ ПРОИЗВОДСТВА
Телодвижения: 4 СОБСТВЕННЫХ завода (Саратов, Энгельс, Петровск, Красноармейск)
Охана маркет: Нет собственного производства (контрактное/закупки)

## 3. ПРОДУКТОВАЯ СТРАТЕГИЯ
Телодвижения: Фокус — спортивная одежда (худи, свитшоты, брюки, футболки)
Охана маркет: Широкий ассортимент (халаты, пижамы, туники, брюки, футболки, кухонные полотенца!)

## 4. ЦЕНОВОЕ ПОЗИЦИОНИРОВАНИЕ
Телодвижения: Средний ценовой сегмент (ориентировочно 1 500—4 000 руб.)
Охана маркет: Низкий ценовой сегмент (ср. чек 732 руб.)

## 5. КЛЮЧЕВЫЕ ФАКТОРЫ УСПЕХА ТЕЛОДВИЖЕНИЙ
1. Собственное производство → низкая себестоимость → высокая маржа
2. Чёткая бренд-идентичность (российский спортивный бренд)
3. Узкий фокус ниши → легче удерживать позиции в алгоритме WB
4. Более высокий ср.чек → меньше продаж нужно для той же выручки
5. Визит Бакальчук на производство → PR признание, буст алгоритма WB
6. Лонгтейл-история (основан 2016, WB ~2020) → накопленный рейтинг

## 6. ПРОБЛЕМЫ ОХАНА МАРКЕТ
1. Высокий % отмен (27,1%) → сигнал алгоритму WB о проблемах
2. Очень низкий ср.чек (732 руб.) → низкая маржа на единицу
3. Высокая комиссия WB (34-38%) + логистика съедает маржу
4. Размытый бренд (нет единой идентичности, смешанный ассортимент)
5. Конкуренция в низком ценовом сегменте жёсткая (много игроков)
6. 511 SKU но без фокуса — сложно поддерживать качество контента/рейтингов
