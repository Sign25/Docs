## Термины и определения

### A-E

| Термин | Определение |
|--------|-------------|
| **Access Level** | Уровень доступа к документу (staff, manager, senior, director, administrator) |
| **AI Agent** | Сервис Timeweb для выполнения LLM-запросов (GPT-5 mini) |
| **All** | Значение brand_id для документов, доступных всем брендам |
| **Chunking** | Процесс разбиения документа на фрагменты для индексации |
| **Embedding** | Векторное представление текста для семантического поиска |
| **ETL Daemon** | Сервис обработки документов (Extract-Transform-Load) |

### F-K

| Термин | Определение |
|--------|-------------|
| **Filter** | Правила ограничения доступа к документам |
| **GPT-5 mini** | Модель LLM, используемая для генерации ответов |
| **Indexing** | Процесс добавления документа в поисковый индекс |
| **KB (Knowledge Base)** | База знаний, хранилище документов |

### L-P

| Термин | Определение |
|--------|-------------|
| **LLM** | Large Language Model — большая языковая модель |
| **Markdown** | Формат разметки текста, используемый для хранения документов |
| **Moderation** | Процесс проверки документа перед публикацией |
| **OCR** | Optical Character Recognition — распознавание текста на изображениях |
| **OpenSearch** | Поисковый движок, используемый в Timeweb KB |
| **Pipeline** | Последовательность этапов обработки (запроса или документа) |
| **Prompt** | Текст запроса или инструкции для LLM |
| **Prompt-инъекция** | Добавление инструкций в system prompt для управления поведением LLM |

### Q-T

| Термин | Определение |
|--------|-------------|
| **Quarantine** | Карантин для документов, не прошедших валидацию |
| **Query** | Запрос пользователя к базе знаний |
| **RAG** | Retrieval-Augmented Generation — генерация ответа с предварительным поиском |
| **Relevance Score** | Оценка релевантности найденного документа (0.0 — 1.0) |
| **Semantic Search** | Семантический поиск по смыслу, а не по ключевым словам |
| **System Prompt** | Инструкции для LLM, определяющие его поведение |
| **Timeweb KB** | Managed-сервис Timeweb для хранения и поиска документов |
| **Top-K** | Количество наиболее релевантных результатов поиска |

### U-Z

| Термин | Определение |
|--------|-------------|
| **Vector** | Числовое представление текста для поиска |
| **YAML** | Формат метаданных в заголовке документа |

## Значения полей

### access_level

| Значение | Описание |
|----------|----------|
| `staff` | Доступ для всех сотрудников |
| `manager` | Доступ для Manager и выше |
| `senior` | Доступ для Senior и выше |
| `director` | Доступ для Director и выше |
| `administrator` | Доступ только для Administrator |

### brand_id

| Значение | Описание |
|----------|----------|
| `ohana_market` | Бренд Охана Маркет |
| `ohana_kids` | Бренд Охана Кидс |
| `all` | Общие документы для всех брендов |

### category

| Значение | Описание |
|----------|----------|
| `product` | Информация о товарах |
| `regulation` | Регламенты и инструкции |
| `contract` | Договорная документация |
| `finance` | Финансовые документы |
| `analytics` | Аналитика |
| `hr` | Кадровые документы |
| `logistics` | Логистика |
| `marketing` | Маркетинг |
| `technical` | Техническая документация |
| `other` | Прочее |

### status (documents)

| Значение | Описание |
|----------|----------|
| `pending` | Ожидает модерации |
| `approved` | Одобрен |
| `rejected` | Отклонён |
| `indexed` | Загружен в KB |
| `quarantine` | В карантине |
