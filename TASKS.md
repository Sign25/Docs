## 1. Ролевая система

### 1.1 Иерархия ролей (РЕАЛИЗОВАНО)

```
pending (0) → staff (1) → manager (2) → senior (3) → director (4) → admin (5)
```

| Роль | Уровень | Описание |
|------|---------|----------|
| pending | 0 | Ожидание одобрения |
| staff | 1 | Базовый сотрудник |
| manager | 2 | Менеджер (ограничен brand_id) |
| senior | 3 | Старший менеджер |
| director | 4 | Директор |
| admin | 5 | Администратор |
