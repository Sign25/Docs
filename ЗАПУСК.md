# Как запустить Open WebUI

## РАБОЧИЙ СПОСОБ (проверен):

1. **Фронтенд** (в корне):
```bash
$env:VITE_SKIP_PYODIDE = 'true'; npx vite dev --host --port 5173
```
**URL:** http://localhost:5173/

2. **Backend** (из папки backend):
```bash
cd backend
.\venv\Scripts\activate
$env:CORS_ALLOW_ORIGIN='http://localhost:5173;http://localhost:8080'
python -m uvicorn open_webui.main:app --host 0.0.0.0 --port 8080
```

## ВАЖНО:
- ВСЕГДА запускать backend из папки backend с активированным venv
- ВСЕГДА запускать фронтенд из корня проекта\
